使用sar工具查看系统瓶颈

安装sar工具及enable

  • 安装sar工具

    yum install sysstat

  • 修改配置打开sar

  • 查看crontab的内容

使用sar工具

  • 查看cpu使用率

    sar -u

  • 看平均负载情况

    sar -q

  • 查看内存负载

    sar -r

  • 查看内存页面交换情况

    sar -W

  • 查看内存的情况

    sar -B

  • 查看IO的情况

    sar -b
